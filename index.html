HOW TO PLAY

  1. Simon will give the first signal. (One of the Color Lenses will light up, accompanied by a tone.)
  2. Repeat that signal by touching the same Color Lens.
  3. Simon will repeat the first signal and then add one more.
  4. Repeat those two signals in the same order.
  5. Simon will continue to play the sequence and add one. Continue to repeat Simon for as long as you can!
